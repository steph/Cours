Description

phpMyAdmin (PMA) est une application Web de gestion pour les systèmes de gestion de base de données MySQL réalisée en PHP et distribuée sous licence GNU GPL.

Crée une Base de données avec phpMyAdmin

  1. Sélectionner l'onglet Base de données
  2. Insérer le nom de votre BDD dans le champs de saisie
  3. Créer

Sécurité

Il est recommandé de crée un utilisateur par BDD.

  1. Sélectionner l'onglet Privilèges
  2. Cliquez sur le lien Ajouter un utilisateur
  3. Saisir le Nom d'utilisateur
  4. Saisir le client* "Local"
  5. Saisir un Mot de passe
  6. Sélectionner "Crée une base portant ..."

Crée une Table avec phpMyAdmin

  1. Sélectionner votre BDD
  2. Cliquez sur Nouvelle table ou
    • Saisir le nom de la table
    • Saisir le nombre de colonne
    • Exécuter

Création de table -suite-

  1. Saisir le nom de la table
  2. Saisir les données pour une colonne
  3. Sélectionner le Moteur de stockage
  4. Sélection l'interclassement
  5. Si besoin rajouter des colonnes
  6. Exécuter

Faire des requètes SQL

Pour faire des requètes SQL, Sélectionner votre BDD puis l'onglet SQL. Les requètes effectué seront faites sur la BDD choisi.
